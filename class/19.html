Color Properties

  • color

Color Values

  • By Name
  • By RGB
  • By Hexidecimal
h1 {
    color: rgb(20%, 20%, 20%);  /* Red Green Blue */
    
    color: silver;              /* Named color */
    
    color: #0a0a0a;             /* Hex - Red Green Blue */
}

Hexidecimal Values

#01ffa5   -- Hex value

R -- 01 (1)
G -- ff (255)
B -- a5 (165)

Hexidecimal Colors

#ffffff  -- white
#000000  -- black
#080808  -- gray

#ff0000  -- red
#00ff00  -- green
#0000ff  -- blue

Box Model

Margin is outside; Padding is inside

    padding: 50px;
    margin: 10px;
    border: solid 1px gray;
}
