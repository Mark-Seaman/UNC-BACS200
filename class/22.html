Rule #22 - Add a menu to navigate through your side

Main Menu

  • Can go into the "
    " or above
  • Your choice

nav Element

Put the nav element in the header


<header>
    <nav></nav>
</header>

Nav Element

  • The nav element defines special areas of the page content
<nav>
    <ul>
        <li><a href="/">Home</a></li>
        <li><a href="project">Projects</a></li>
    </ul>
</nav>
<header>My Site Title</header>
<main>
    <h1>My Home Page</h1>
    
    <p>Paragraph</p>
</main>

Menu Items

Main menu is a list of hyperlinks

<nav>
    <ul>
        <li>Menu Item 1</li>
        <li>Menu Item 2</li>
        <li>Menu Item 3</li>
    </ul>
</nav>

List Style

CSS Property - list-style-type

nav ul {
    list-style-type: none;
    padding: 0em 1.5em;
    text-align: center;
}

List Items

CSS Property - display

nav ul li {
    display: inline;
    padding: 0em 1.5em;
    width: 200px;
    background-color: blue;
    color: white;
    box-shadow: 2px 2px 5px gray;
}

List Links

nav ul li a {
    color: yellow;
    font-weight: bold;
    padding: 2em;
}

Side Menu

<main>
    <nav>
        <ul>
            <li><a href="home.html">Home</a></li>
            <li><a href="about.html">About Us</a></li>
            <li><a href="details.html">Details</a></li>
        </ul>
    </nav>

    <section class="page">
        <h1>Main</h1>
        <p>Main section</p>
    </section>
</main>

List Style

CSS Property - list-style-type

nav {
    margin: 10px;
    float: left;
    background-color: lightyellow;
    box-shadow: 2px 2px 5px gray;
    text-align: center;
}
nav ul {
    list-style-type: none;
}

List Items

CSS Property - display

nav ul li {
    padding: 1em;
    width: 200px;
    margin: 10px;
    background-color: lightblue;
    box-shadow: 2px 2px 5px gray;
}

List Links

nav ul li a {
    display: block;
    color: black;
    font-weight: bold;
    text-decoration: none;
}
